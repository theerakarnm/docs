### Body

<ParamField
  required
  body='provider'
  type='EMAIL | FACEBOOK | GOOGLE'>
  The provider of the user. This can be EMAIL, FACEBOOK or GOOGLE.
</ParamField>

<ParamField
  required
  body='email'
  type='string'>
  The email of the user. This is required if the provider is EMAIL.
</ParamField>

<ParamField
  body='password'
  type='string'>
  Password of user if provider is EMAIL, and password must be at least 6 characters long.
</ParamField>

<ParamField
  body='name'
  type='string'>
  The user name.
</ParamField>

<ParamField
  body='surname'
  type='string'>
  The user surname.
</ParamField>

<ParamField
  required
  body='phone'
  type='string'></ParamField>

<ParamField
  required
  body='role'
  type='CLIENT | PARTNER'>
  The role of the user. This can be CLIENT or PARTNER.
</ParamField>
