---
title: 'Login'
api: 'POST https://pgvx2y0z67.execute-api.ap-southeast-1.amazonaws.com/dev/api/v1/auth/login'
description: 'login to account'
---

### Body

<ParamField
  body='email'
  type='string'
  required>
  This is the email address of the user you want to login as.
</ParamField>

<ParamField
  body='password'
  type='string'
  required>
  This is the password of the user you want to login as.
</ParamField>

<ParamField
  body='provider'
  type='EMAIL | FACEBOOK | GOOGLE'
  required>
  This is the provider of the user you want to login as.
</ParamField>

### Response

<ResponseField
  name='message'
  type='string'>
  This is the message returned by the API.
</ResponseField>

<ResponseField
  name='timestamp'
  type='Date'>
  The timestamp of the response.
</ResponseField>

<ResponseField name="data" type="object">

The data of the user that was login.

<Expandable title="Toggle object">

<ResponseField
  name='refreshToken'
  type='string'>
  This is the refresh token (refreshToken or REFRESH_TOKEN) that will be used to refresh the access token. You should save this on your end to use when refreshing the access token.
</ResponseField>

<ResponseField
  name='accessToken'
  type='string'>
  This is the access token (accessToken or ACCESS_TOKEN) that will be used to authenticate the user. You should save this on your end to use when authenticating the user.
</ResponseField>

</Expandable>

</ResponseField>
